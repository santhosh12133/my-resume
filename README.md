# React Resume Website

A modern, responsive, and professional resume website built using React and plain CSS. This project presents my academic background, technical skills, projects, and experience in a clean, recruiter-friendly format.

## Tech Stack
- React (Functional Components)
- Plain CSS (No frameworks)
- JavaScript (ES6+)
- Git & GitHub
- npm

## Features
- Single-page resume layout
- Professional dark theme
- High-contrast, readable typography
- Fully responsive (mobile & desktop)
- ATS-friendly structure
- Easy to customize

## Project Structure


src/
│── App.js
│── Resume.js
│── Resume.css
│── index.js


## Getting Started

### Clone the Repository


git clone https://github.com/santhosh12133/My-resume.git

cd your-repo-name


### Install Dependencies


npm install


### Run the Application


npm start


The application will be available at:
http://localhost:3000

## Customization
- Update resume content in `Resume.js`
- Modify colors and fonts in `Resume.css`
- Adjust layout or sections as needed

## Use Cases
- Personal resume website
- Portfolio for internships and entry-level roles
- MCA academic project
- Base template for future portfolio expansion
